# Installation on Cloudflare Workers

:::info

This is the legacy installation guide. Please use the new [Cloudflare Workers Quickstart](../getting-started/quickstart-cloudflare) for the simplified npm-based installation.

:::

This guide is kept for reference purposes only. The recommended approach is to use the npm packages as described in the quickstart guide.

## Recommended Installation

Use the npm-based installation:

```bash
$ npm init -y
$ npm install @package-broker/cli @package-broker/main
$ npx package-broker
```

See the [Cloudflare Workers Quickstart](../getting-started/quickstart-cloudflare) for the complete guide.
